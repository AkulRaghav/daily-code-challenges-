| Category | Problems |
|----------|----------|

| Sliding Window | Longest Substring, Container With Most Water, Sliding Window Maximum |
| Linked Lists | Merge Two Sorted, Linked List Cycle |
| Trees | Invert Binary Tree, Lowest Common Ancestor |
| Graphs | BFS Shortest Path, Dijkstra's, Topological Sort |
| Dynamic Programming | Climbing Stairs, Best Time Buy/Sell, 0/1 Knapsack |
| Data Structures | HashMap, Trie, LRU Cache, Min-Heap, Union-Find |
| Sorting | Merge Sort |
| Intervals | Merge Intervals |

## Structure

Each day has:
- `problem.md` — problem statement, examples, constraints
- `solution.js` — implementation with detailed comments, complexity analysis, and tests

node solutions/2026-06-23/solution.js
